# learn-ui

[English](./README.md) | 简体中文

## 快速开始
### 安装依赖
```shell
pnpm install
```

### 开发项目
```shell
pnpm dev
```

### 新建组件

```shell
pnpm new
```

*新建一个名为 Button 的函数组件👇*
```shell
pnpm new Button -- -f
```

---

## 构建和运行
### 构建
```shell
pnpm build
```

*构建项目时绕过所有检查👇*
```shell
pnpm build -- -n
```

### 生产环境运行项目
```shell
pnpm start
```

**更多配置项请在 [omni.config.js](https://github.com/omni-door/cli/blob/master/docs/OMNI.zh-CN.md) 中编辑**
